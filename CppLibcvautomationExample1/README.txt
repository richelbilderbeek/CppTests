After successfully building libcvautomation, there will be two executables in [YOURPATH]/libcvautomation/examples.
cva-input will be used by the shell script. 

Copy the script to /libcvautomation/examples:

cp CppLibcvautomationExample1.sh [YOURPATH]/libcvautomation/examples

Run it:

./CppLibcvautomationExample1.sh
